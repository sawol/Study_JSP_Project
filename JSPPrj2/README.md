# <홈페이지 만들기>

## 기간  
2022.3.28~4.13  

## 목표  
- JSP를 이해한다.  
- JDBC를 이해한다.
- 게시판을 만들어본다.
- 이후, Spring을 배우기 위한 기반 지식 쌓기.
